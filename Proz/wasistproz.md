## Was ist eine Prozedur?

Eine Prozedur ist ein grundlegendes Konzept in der Programmierung, das eine Sammlung von Anweisungen zusammenfasst, die eine bestimmte Aufgabe oder Funktion erfüllen. Prozeduren ermöglichen es, wiederverwendbaren Code zu erstellen, der unter einem Namen zusammengefasst ist. Diese Herangehensweise vereinfacht das Programmieren, da man eine Prozedur einmal definieren und dann an verschiedenen Stellen im Programm aufrufen kann, ohne den Code mehrfach schreiben zu müssen. Prozeduren mit Rückgabewerten nennt man auch Funktionen.
